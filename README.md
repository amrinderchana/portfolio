# Personal Portfolio Website

A responsive static HTML portfolio website built to showcase professional experience, projects, skills, and contact information.

## Features

- Modern and responsive design
- Clean single-page layout
- About section
- Skills showcase
- Projects section
- Contact section
- Smooth scrolling navigation
- Mobile-friendly interface
- Lightweight and fast-loading static website

## Tech Stack

Built with:

- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started

To run locally:

1. Clone the repository

```bash
git clone https://github.com/amrinderchana/portfolio.git
```

2. Navigate into the project

```bash
cd portfolio
```

3. Open `index.html` in your browser

Or use VS Code Live Server:

```bash
Right Click → Open with Live Server
```

## Customization

Update the following sections in `index.html`:

- Personal information
- About content
- Skills list
- Project details
- Social links
- Contact information

## Deployment

This project can be deployed easily on:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

### GitHub Pages Deployment

1. Push code to GitHub
2. Go to Repository Settings → Pages
3. Select branch (`main`)
4. Save and publish

Your site will be live in minutes.
